# 自主學習導航系統 (Self-Study Navigation System)
「不是為了塞進更多進度，而是為了讓孩子換取真正的自由。」

這是一款專為自主學習（Homeschooling）設計的管理系統。與傳統「填鴨式」進度表不同，本系統的核心理念是**「主動權移轉」**——透過視覺化的工具，引導孩子從「被動接受安排」轉向「主動規劃生活」。
---

##🌟 核心設計理念 (Design Philosophy)
白紙黑字，自主負責：系統設計強調「手動規劃」，讓孩子在分配時間的過程中學會權衡與取捨。

視覺化進度債務：透過衝刺看板，讓孩子直觀感受進度延遲的壓力，練習面對現實並調整計畫。

從量變到質變：結合 AI 技術，將枯燥的考卷練習轉化為邏輯診斷。
---

## 🛠️ 核心功能 (Core Features)

1. **進度導航看板：**

   月曆任務清單：將實體教材厚度轉化為視覺化時間區塊。
   
   進度衝刺看板：誠實呈現進度延遲與完成率，拒絕「還有明天」的心理陷阱。
   
   成就軌跡看板：記錄學習心得與分數趨勢，建立長期的自我成就感。

2. **AI 智慧輔助（整合 Gemini API）：**

   錯題邏輯診斷：不只是給答案，而是分析孩子的錯誤邏輯並給予引導。
   
   個人化強化練習：AI 根據錯題紀錄，自動生成相似題或延伸練習，達到真正的舉一反三。
   
   數據分析回饋：整合 ECharts，將枯燥的分數轉化為成長曲線，讓進步清晰可見。


---

## 🚀 技術架構 (Tech Stack)

- **前端**: Vue 3.5 (Vite), Element Plus, ECharts (趨勢圖表)
- **後端**: Flask 3.1.2, SQLAlchemy (資料庫管理)
- **AI 引擎**: Google Gemini 1.5 Flash (用於診斷與自動出題)
- **資料庫**: MySQL 8.0+

---

## 🚀 快速安裝手冊

### 1. 資料庫建置
- 建立資料庫：`CREATE DATABASE selfstudy_db;`
- 執行根目錄的 `schema.sql` 以初始化資料表結構。

### 2. 後端環境 (Backend)
- 進入 `backend` 資料夾並建立虛擬環境。
- 執行 `pip install -r requirements.txt` 安裝依賴。
- 於 `.env` 設定資料庫連線資訊後，執行 `python app.py`。

### 3. 前端環境 (Frontend)
- 進入 `frontend` 資料夾。
- 執行 `npm install` 安裝套件。
- 執行 `npm run dev` 啟動系統開發介面。

關於 Gemini AI 功能： 本系統整合了 Gemini AI 進行錯題診斷，為了保障個人隱私並由使用者自行控管 API 額度，系統不預存 API Key。

設定方式：請使用者至系統內的「設定頁面」，輸入您個人的 Gemini API Key 即可啟用 AI 診斷與出題功能。您的 Key 將儲存在本地端（或瀏覽器中），不會上傳至我們的伺服器。
---

## 📂 專案結構
